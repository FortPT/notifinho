# Product icons

These self-contained 128 px PNG badges are used as Discord embed thumbnails
and Microsoft Teams Adaptive Card header images. They use product initials
and familiar brand colors so delivery does not depend on a third-party image
CDN. Xen Orchestra keeps its existing Vates-hosted logo.

The editable SVG sources for the v1.9.0 Redfish, Supermicro, HPE iLO,
Dell iDRAC, and Home Assistant badges are stored in `source/`. Release PNGs
are rendered at 128 px so Discord and Teams receive a consistent lightweight
asset.

Product names and logos remain trademarks of their respective owners. They
are used only to identify the notification source.
