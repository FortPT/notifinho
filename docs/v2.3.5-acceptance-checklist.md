## Source removal

- [ ] An inactive source displays Remove for an administrator.
- [ ] Cancelling the confirmation leaves it unchanged.
- [ ] Confirming removal sends a DELETE request with the source in the URL path.
- [ ] The request has no JSON body requirement.
- [ ] The source disappears from Sources and Overview.
- [ ] Historical delivery records remain available.
- [ ] An enabled exact route still returns HTTP 409 and blocks removal.
- [ ] An enabled wildcard route alone does not block removal.
