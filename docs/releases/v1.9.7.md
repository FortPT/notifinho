# Notifinho v1.9.7 release notes

## Release summary

Notifinho v1.9.7 permanently fixes the official-logo regression discovered
after the v1.9.6 stable image replaced the accepted preview images. The stable
build now embeds an immutable asset URL for its exact release commit, and
Discord Components V2 delivery uploads the packaged official PNG instead of
depending only on a remote fetch.

The approved v1.9.6 Teams and Discord layouts, source timestamps, routes,
aliases, filters, endpoints, and secrets are unchanged.

## Fixed

- The Docker release workflow passes the checked-out tag commit's immutable
  `assets/icons` URL into the image through `NOTIFINHO_ICON_BASE_URL`.
- The Dockerfile preserves that release-specific URL in the final runtime
  environment, so Teams never falls back to mutable `main` or initials badges.
- Discord output finds both legacy embed thumbnails and Components V2 type-11
  thumbnail media, rewrites the selected URL to `attachment://...`, and sends
  the packaged PNG using multipart webhook delivery.
- Automated coverage validates the build-time URL contract and inspects the
  effective Components V2 attachment payload and bytes.

## Upgrade from v1.9.6

1. Back up the production configuration and stack definition.
2. Pull `fortpt/notifinho:1.9.7` or `ghcr.io/fortpt/notifinho:1.9.7` after the
   release workflow succeeds.
3. Change only the image tag and recreate the container with all existing
   mounts, ports, routes, aliases, filters, webhooks, and secrets preserved.
4. Confirm `NOTIFINHO_ICON_BASE_URL` contains the v1.9.7 release commit rather
   than `main`.
5. Deliver one real Teams and one real Discord event and visually confirm the
   official vendor logo.

No stack environment variable or configuration migration is required.

## Rollback

Return temporarily to `fortpt/notifinho:1.9.6-icons-hotfix-63ce514` while
preserving the same configuration and secrets.

## Release gate

- Complete automated test suite passes.
- Official Docker image contains all 17 packaged PNGs.
- Runtime icon base equals the immutable v1.9.7 release commit URL.
- A real Teams and Discord notification display official icons before the
  temporary v1.9.6 hotfix image is retired.
