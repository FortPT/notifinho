# Notifinho v1.5.2 release notes

## Release summary

Notifinho v1.5.2 improves the presentation of native UniFi Protect
notifications in Discord and Microsoft Teams.

No configuration or routing changes are required.

## UniFi Protect presentation improvements

Protect notification cards now:

- use the actual Protect trigger as the visible title;
- convert identifiers such as `admin_access` to `Admin Access`;
- convert known device labels such as `nvr` to `NVR`;
- preserve the configured Alarm Manager rule in a dedicated `Alarm rule` field;
- avoid incomplete condition text such as `admin_access is`;
- omit redundant conditions that repeat the trigger type;
- retain privacy filtering for MAC addresses, UUIDs, placeholder identifiers,
  and opaque device tokens.

For example:

📹 ℹ️ Admin Access

Admin Access detected by NVR

🎯 Trigger type

🚨 Alarm rule
